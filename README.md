提示词参考： 
用Three.js创建一个实时交互的3D粒子系统。 要求：
 1.通过摄像头检测双手张合控制粒子群的缩放与扩散；
2.提供UI面板可选择爱心/花朵/土星/佛像/烟花等模型；
3.支持颜色选择器调整粒子颜色；
4.粒子需实时响应手势变化；
5.界面简洁现代，包含全屏控制按钮。

角色设定：
你是一位精通 Three.js、WebGL和计算机视觉的创意前端开发专家。
任务目标：
编写一个包含 HTML、CSS和
JavaScript的单文件应用。该应用需要结合Three.js和MediaPipe
Hands来创建一个基于手势控制的3D粒子+照片云构成的圣诞树。
交互逻辑；圣诞树有合拢态、散开态，照片放大态。
手势动作：据拳=合拢；五指张开=散开；手旋转=旋转画面；抓取=放大照片。
交互要求：状态之间平滑切换，带有过渡动画。
技术备忘：
可能用到的技术栈：Three.js、-WebGL. MediaPipe Hands.
